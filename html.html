✅ HTML শেখার জন্য যা যা দরকার (Step-by-Step Guide) 1️⃣ HTML কী? HTML (HyperText Markup Language) হলো ওয়েবপেজ তৈরি করার একটি ভাষা। এটা দিয়ে আমরা ব্রাউজারে দেখা যায় এমন টেক্সট, ইমেজ, লিঙ্ক, বাটন, লেআউট ইত্যাদি তৈরি করি। 2️⃣ HTML শিখতে যেসব টুল দরকার ✔ একটি Text Editor HTML কোড লেখার জন্য এগুলোর যেকোনো একটি ব্যবহার করতে পারো: VS Code (সেরা) Sublime Text Notepad++ এমনকি সাধারণ Notepad হলেও চলবে ✔ একটি Web Browser HTML ফাইল রান/দেখার জন্য: Google Chrome Firefox Edge Brave ✔ একটি ভালো ফোল্ডার স্ট্রাকচার ওয়েবপেজ বানানোর সময় সাধারণত ফোল্ডারগুলো এইভাবে সাজানো হয়: project/ ├─ index.html ├─ about.html ├─ style.css ├─ script.js └─ images/ 3️⃣ HTML এর বেসিক স্ট্রাকচার প্রতিটি HTML ডকুমেন্টের গঠন এমন হয়: My First Webpage
